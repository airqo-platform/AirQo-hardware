# Maintenance Toolkit

This page lists the tools and equipment required to perform maintenance on the AirQo Binos Air Monitor.

---

## Required Tools

| Tool | Purpose |
|---|---|
| Screwdriver set (Phillips & flathead) | Opening enclosure, tightening mounting hardware |
| Adjustable spanner / wrench | Mounting pole hardware |
| Multimeter | Checking battery voltage, power rails |
| Micro SD card reader | Reading/replacing SD card data logs |
| USB cable (Micro USB) | Firmware updates, serial diagnostics |
| Soft brush / compressed air | Cleaning sensor inlet and solar panel |
| Laptop with Arduino IDE / PlatformIO | Firmware flashing and diagnostics |

---

## Consumables

| Item | Notes |
|---|---|
| LiPo 3.7V 3.5AH battery | Solar-powered unit replacement battery |
| Li-ion 3.7V 8.8AH battery | Mobile unit replacement battery |
| Micro SD card (Class 10, ≤32GB) | Data logging card replacement |
| SIM card | GSM connectivity |
