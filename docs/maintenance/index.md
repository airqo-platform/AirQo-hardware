# Maintenance

The Binos Air Monitor requires routine maintenance to enhance its uptime and efficiency. Device failures or inefficiencies in a large interconnected network of monitors can have a significant impact on data quality and platform reliability.

---

| Area | Description |
|---|---|
| [Maintenance Toolkit](toolkit.md) | Tools and equipment needed for maintenance activities |
| [Device Diagnosis](diagnosis.md) | Diagnosing common device faults and failures |
| [System Maintenance](system.md) | Scheduled system-level maintenance procedures |

---

## Maintenance Schedule

| Frequency | Activity |
|---|---|
| **Monthly** | Visual inspection, clean sensor inlet, check mounting, clean solar panel |
| **Quarterly** | Check battery health, verify data transmission, check SD card storage |
| **As needed** | Firmware update, replace faulty sensors, battery replacement |
